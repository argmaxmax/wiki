Definition: Agebraic Interior Point

Let X be an ℝ-Vectorspace, and M ⊆ X. Then x ∈ M is called an algebraic interior point of M if ∀y ∈ M there is some ε > 0 such that x + ty ∈ M ∀t ∈ ( − ε, ε).

Remark:

If X is a normed space and x ∈ M̊, then x is an algebraic interior point. In fact, this the case for any topological vector space.

Theorem:

Let X be an ℝ-Vectorspace, $C\subseteqX$ convex with an algebraic interior point.
Then there exists some functional F ∈ X′ such that
F(x) ≥ F(z)  ∀z ∈ C
and
F(x) > f(z)  ∀ algebraic interior points of C.

Proof.

Translation of Problem: Wlog 0 ∈ C is an algebraic interior point of C. (By Convexity stability under affine transformations.)
Then define p : X → ℝ by
$$ p(x) = \inf \left\{ t \gt 0 \mid \frac{x}{t}\in C \right\} $$

Then p is a sublinear functional. ###### Proof: Homogeneity is obvious. For x, y ∈ X let s, t > 0 be such that $\frac{x}{s}\in X,$ $\frac{y}{t} \in c$ and it follows
$$ \frac{x+y}{s+t} = \frac{s}{s+t}\frac{x}{s}+ \frac{x}{s+t}\frac{y}{t} $$
which implies s + t ≥ p(x + y). Taking the infima over (s, t) satifying ( * ) yields the subadditivity.
Set S := ℝx. Define
$$ f:\begin{cases}S &\to &\mathbb R \\ \lambda x &\mapsto &f(\lambda x):=\lambda p(x) \end{cases} $$
for λ ∈ ℝ.
By Hahn-Banach ∃F ∈ X′ such that F|S = f and F(y) ≤ p(y) for λ ∈ X.
For z ∈ C it holds
F(z) ≤ p(z) ≤ 1 ≤ p(x) = F(x).

For z ∈ C an algebraic interior point of C it holds
$$F(z)\ltF(x) $$
since p(z) < 1 for algebraic interior points because $\frac{z}{t}\in C$ for some t < 1. ◼

Satz: Lipschitz-Stetigkeit beschränkter Operatoren

Sei T ∈ B(X, Y) ein beschränkter Operator auf normierten 𝕂-Vektorräumen. Dann ist T Lipschitz-stetig.
Details. Da ∥Tx∥ ≤ C ⋅ ∥x∥ für alle x ∈ X für ein C > 0, ist
∥Tx−ty∥ = ∥T(x−y)∥ ≤ C ⋅ ∥x−y∥
für x, y ∈ X.◼

Satz: Vektorräume der Linearformen und Linearen Abbildungen

Seien X1, …, Xp K-Vektorräume. Dann ist die Menge aller Multilinearformen 𝒥p(X1, …, Xp) ein Vektorraum. Weiterhin ist hom (X1, X2) ein Vektorraum. Als Spezialfall ist der Dualraum X* ein Vektorraum. Ist K = 𝕂 so sind die Operatormengen ℒ(X1, X2), B(X1, X2) Vektorräume.

Satz:

Seien X, Y normierte Räume und T ∈ B(X, Y) ein beschränkter Operator. Dann ist die Operatornorm ∥⋅∥ eine Norm auf B(X, Y) und es gilt für alle x ∈ X:
∥Tx∥ ≤ ∥T∥ ⋅ ∥x∥

Ist weiterhin Z ein weiterer normierter Raum und S ∈ B(Y, Z) ein weiterer beschränkter Operator, so gilt für die Komposition:
∥ST∥ ≤ ∥S∥ ⋅ ∥T∥.

Details. Dadurch dass die Operatornorm ∥⋅∥ auf B(X, Y) endlich ist, bildet sie in 𝕂 ab. Die Definitheit folgt daraus, dass die Operatornorm ein Supremum aus nichtnegativen Werten ist, welches also genau dann Null ist wenn jeder dieser Werte null ist, also muss, da der Kern eines Operators immer die Null enthält und der Nenner nichtnegativ ist, T = 0 sein. Ist α ∈ 𝕂, so folgt die Absolute Homogenität (für x ≠ 0) wegen
$$ \frac{\left\lVert \alpha T x \right\rVert }{\left\lVert x \right\rVert } = \left| \alpha \right| \frac{\left\lVert Tx \right\rVert }{x} $$
aus der positiven Homogenität des Supremums. Weiterhin gilt die Dreiecksgleichung wegen der Additivität des Supremums und
$$ \frac{\left\lVert (T+S)x \right\rVert }{\left\lVert x \right\rVert } = \frac{\left\lVert Tx+Sx \right\rVert }{\left\lVert x \right\rVert }\le \frac{\left\lVert Tx \right\rVert }{\left\lVert x \right\rVert }+\frac{\left\lVert Sx \right\rVert }{\left\lVert x \right\rVert } $$

Die Ungleichung für den Funktionswert folgt direkt aus
$$ \frac{\left\lVert Tx \right\rVert}{\left\lVert x \right\rVert }\le \displaystyle \sup_{x\in X\\x\neq 0} \frac{\left\lVert Tx \right\rVert }{\left\lVert x \right\rVert }. $$

Kommentar:
